# Cloudflare Tunnel

This folder contains repo-local templates for exposing the app on `http://localhost:3000`.

## Files

- `config.example.yml`: template for a named tunnel
- `config.yml`: local real config, intentionally gitignored
- `*.json`: tunnel credentials generated by Cloudflare, intentionally gitignored if stored here

## Intended flow

1. Run `npm run cf:tunnel:login`
2. Run `npm run cf:tunnel:create`
3. Copy `config.example.yml` to `config.yml`
4. Fill in:
   - tunnel ID
   - hostname
   - Windows username
5. Run `npm run cf:tunnel:validate`
6. Run `npm run cf:tunnel:run`

## Service target

The tunnel forwards traffic to:

```text
http://localhost:3000
```
